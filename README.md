# utility

Docker image containing various tools

## What's installed

> [!NOTE]
> By default, this image is built for macOS runing on their silicon specified by the `ARCH` argument.\
> You can easily change it by supplying `--build-arg ARCH=[whatever]`.

- Open Tofu
- Terraform Switch
- Terraform Docs
- AWSCLI v2
- jq
- yq
- kubectl
- helm
